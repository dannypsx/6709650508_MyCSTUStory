MY JOURNEY

ปีที่ 1

การปูพื้นฐานเข้าใจพื้นฐานของการเขียนโปรแกรม และการพัฒนาทักษะทางคณิตศาสตร์ การเขียนแอปพลิเคชัน เริ่มฝึกฝนการทำงานเป็นทีมและการสื่อสารผ่านโปรเจกต์เล็กๆ

ปีที่ 2

การพัฒนาทักษะเชิงลึก เข้าใจการออกแบบและพัฒนา Software รวมถึงทักษะด้านการแก้ปัญหาเชิงวิศวกรรม ทำโปรเจกต์ขนาดเล็กกับเพื่อนร่วมทีมฝึกการใช้เครื่องมือการพัฒนา Software เช่น Git, IDE, และ Cloud Computing พัฒนาความเข้าใจเชิงลึกเกี่ยวกับวิธีการทำงานของระบบคอมพิวเตอร์และโครงสร้างข้อมูลที่ซับซ้อน เริ่มต้นทำงานเป็นทีมในโปรเจกต์จริง

ปีที่ 3

การลงมือทำจริงและฝึกทักษะเชิงปฏิบัติ ใช้ความรู้ในการพัฒนาโปรเจกต์จริง ฝึกทักษะการแก้ปัญหาและทำงานร่วมกับผู้อื่น ฝึกงานกับบริษัทเทคโนโลยีเพื่อเพิ่มประสบการณ์จริง ทำโปรเจกต์หรือวิจัยเล็ก ๆ ที่เกี่ยวข้องกับเทคโนโลยีที่สนใจ เช่น AI, Data Science, Cloud เริ่มศึกษาการพัฒนาทักษะ Soft Skills เช่น การนำเสนอ การทำงานเป็นทีม การบริหารเวลา

ปีที่ 4
