MY FUTURE

ปีที่ 1: การปูพื้นฐาน เป้าหมาย: เข้าใจพื้นฐานของการเขียนโปรแกรม และการพัฒนาทักษะทางคณิตศาสตร์ Introduction to Computer Science Programming 101 (C, Python, or Java) Discrete Mathematics Calculus การเขียนรายงานเชิงวิทยาศาสตร์และภาษาอังกฤษ การเขียนแอปพลิเคชัน เริ่มฝึกฝนการทำงานเป็นทีมและการสื่อสารผ่านโปรเจกต์เล็ก ๆ เป้าหมายหลัก: พัฒนาความสามารถในการเขียนโค้ดและเข้าใจโครงสร้างข้อมูลขั้นพื้นฐาน เริ่มรู้จักกับชุมชนวิชาการและเทคโนโลยี
ปีที่ 2: การพัฒนาทักษะเชิงลึก เป้าหมาย: เข้าใจการออกแบบและพัฒนา Software รวมถึงทักษะด้านการแก้ปัญหาเชิงวิศวกรรม วิชาที่เน้น: Data Structures and Algorithms Object-Oriented Programming (OOP) Computer Architecture Operating Systems Software Engineering กิจกรรมนอกหลักสูตร: ทำโปรเจกต์ขนาดเล็กกับเพื่อนร่วมทีม หรือเข้าร่วมการประกวดแข่งขัน Hackathon เข้าร่วมสโมสรนักศึกษา และงาน Networking เพื่อเริ่มสร้างเครือข่ายทางวิชาชีพ ฝึกการใช้เครื่องมือการพัฒนา Software เช่น Git, IDE, และ Cloud Computing เป้าหมายหลัก: พัฒนาความเข้าใจเชิงลึกเกี่ยวกับวิธีการทำงานของระบบคอมพิวเตอร์และโครงสร้างข้อมูลที่ซับซ้อน เริ่มต้นทำงานเป็นทีมในโปรเจกต์จริง ปีที่ 3: การลงมือทำจริงและฝึกทักษะเชิงปฏิบัติ เป้าหมาย: ใช้ความรู้ในการพัฒนาโปรเจกต์จริง ฝึกทักษะการแก้ปัญหาและทำงานร่วมกับผู้อื่น วิชาที่เน้น: Database Management Systems Artificial Intelligence / Machine Learning (เลือกตามความสนใจ) Networks and Security Human-Computer Interaction (HCI) Electives in Specialized Fields (เลือกเรียนวิชาที่สนใจ เช่น Web Development, Mobile Apps, Data Science) กิจกรรมนอกหลักสูตร: ฝึกงานกับบริษัทเทคโนโลยีเพื่อเพิ่มประสบการณ์จริง ทำโปรเจกต์หรือวิจัยเล็ก ๆ ที่เกี่ยวข้องกับเทคโนโลยีที่สนใจ เช่น AI, Data Science, Cloud เริ่มศึกษาการพัฒนาทักษะ Soft Skills เช่น การนำเสนอ การทำงานเป็นทีม การบริหารเวลา เป้าหมายหลัก: สร้างประสบการณ์ในโลกจริงผ่านการฝึกงานและโปรเจกต์ฝึกหัด เริ่มต้นมองหาเส้นทางอาชีพที่สนใจ ปีที่ 4: การเตรียมความพร้อมสู่อนาคต เป้าหมาย: เตรียมความพร้อมเข้าสู่ตลาดงานหรือศึกษาต่อในระดับสูง วิชาที่เน้น: Advanced Software Development Capstone Project (โปรเจกต์ขนาดใหญ่ที่จะใช้ทักษะทุกอย่างที่ได้เรียนรู้) Computer Science Electives (เลือกวิชาที่เสริมทักษะเฉพาะที่ต้องการ เช่น Cybersecurity, Blockchain, Game Development) Seminar in Professional Development กิจกรรมนอกหลักสูตร: เตรียม Portfolio และ Resume สำหรับการสมัครงาน หางานฝึกงานหรือโปรเจกต์เสริมเพื่อสร้างประสบการณ์เพิ่มเติม เริ่มทำงาน Freelance หรือโปรเจกต์ส่วนตัว เพื่อเสริมทักษะการทำงานจริง เข้าร่วมการสัมมนาและงาน Networking ที่เชื่อมโยงกับอุตสาหกรรมที่สนใจ เป้าหมายหลัก: พัฒนาความเชี่ยวชาญในทักษะที่สนใจ สร้างโปรไฟล์ที่โดดเด่น และเตรียมความพร้อมสำหรับการเข้าสู่โลกการทำงาน
